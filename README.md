## Getting started

1. Download the project by pressing [this link here](https://github.com/Vatis-Tech/asr-client-js-html-js-example/archive/refs/heads/main.zip).
2. Unzip the downloaded content.
3. Or, you can clone it with the following command `git clone https://github.com/Vatis-Tech/asr-client-js-html-js-example.git`
4. Go inside `assets/js/index.js`, and replace `YOUR_API_KEY_HERE`, with a valid **API Key** from https://vatis.tech/account/api-key.
5. Open `index.html` in the browser of your choice.

## Notes

The version of `@vatis-tech/asr-client-js` used in this example, is **v1.2.1**.
